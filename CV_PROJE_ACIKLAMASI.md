# CV Proje Açıklaması

ASP.NET Core 9.0, CQRS, MediatR ve Entity Framework Core kullanarak microservice mimarisinde bir e-ticaret uygulaması geliştirdim. FluentValidation, AutoMapper ve Global Exception Handler ile clean code prensiplerine uygun bir mimari tasarladım. PostgreSQL ile Database per Service pattern'ini uygulayarak her servisin bağımsız veritabanına sahip olmasını sağladım. Redis ile Cache-aside pattern implementasyonu yaparak yüksek performans elde ettim. RabbitMQ ve MassTransit ile event-driven mimari kurarak asenkron mesajlaşma, gRPC ile senkron iletişim sağladım. YARP ile API Gateway pattern'ini uygulayarak merkezi routing sağladım. Docker ve Docker Compose ile tüm sistemi containerize ederek tek komutla çalıştırılabilir hale getirdim.
